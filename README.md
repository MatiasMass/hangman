Play the classic Hangman Game on Python!

### How to Play

1. Use the left click to select the letter.

### Requirements

- Python 3.x
- Pygame library

### Installation

1. Clone the repository:

   ```
   git clone https://github.com/MatiasMass/hangman-python.git
   ```
2. Install the required dependencies:

```
pip install pygame
```

3. Launch the game:
```
python main.py
```
